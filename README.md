### A Machine Learning Implementation to Predict Customer Churn

Given real data from a certain SaaS business, we attempt to preduct customer churn.
  
Within the source are implementations using:  
* Neural Nets
* SVMs
* Decision Trees
* Random Forests
* XG Boost
